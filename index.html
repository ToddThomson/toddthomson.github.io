Mila

Achilles Mila Deep Neural Network library provides a comprehensive API to model, train and evaluate Deep Neural Networks for both research and production environments. Features state-of-the-art architectures including transformers, convolutional networks, and recurrent models with NVIDIA CUDA GPU acceleration.

  • GPT2 and Recurrent Neural Networks
  • NVIDIA CUDA runtime GPU acceleration
  • Distributed training across multiple GPUs
  • Optimized data processing pipelines
  • Mixed precision training support
  • Docker containerization for Linux
